# Микросервис Arctic Team

Микросервис на Python + **FastAPI** для мониторинга и анализа физиологических, нейрофизиологических и продуктивностных показателей участников экспедиций в экстремальных условиях.  
Получает данные из PostgreSQL, строит графики и предоставляет ИИ-анализ и рекомендации через **GigaChat**.

**Основные возможности:**
- Получение метрик мозга, физиологических, кардио и продуктивности
- Построение визуализаций (графики мозговых волн, стресса, концентрации, расслабления и др.)
- ИИ-анализ состояния участника с рекомендациями (GigaChat)

## Требования

- Python 3.12+
- PostgreSQL
- Ключ авторизации GigaChat
- Docker

## Конфигурация

Все настройки задаются через переменные окружения.

| Переменная            | Описание                        |
|-----------------------|---------------------------------|
| POSTGRES_DB           | Имя базы данных                 |
| POSTGRES_USER         | Пользователь PostgreSQL         |
| POSTGRES_PASSWORD     | Пароль                          |
| POSTGRES_HOST         | Хост базы данных                |
| POSTGRES_PORT         | Порт PostgreSQL                 |
| AUTHORIZATION_KEY     | Ключ авторизации GigaChat API   |

## Доступ к сервису

|-----------|-----------------------|
